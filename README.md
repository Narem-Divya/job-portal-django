# Job Portal Django

## Project Description
This is a Job Portal web application developed using Django. It allows users to register, log in, view available jobs, and apply for jobs online.

## Features
- User Registration
- User Login Authentication
- View Job Listings
- Apply for Jobs
- Success Confirmation Page
- SQLite Database Integration

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## Project Structure
- register.html – User Registration
- login.html – User Login
- home.html – Home Page
- jobs.html – Job Listings
- apply.html – Job Application Form
- success.html – Application Success Page

## How to Run
1. Clone the repository
2. Install Django
3. Run:
   python manage.py runserver
4. Open:
   http://127.0.0.1:8000/
